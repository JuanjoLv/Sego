Descripción:


Debido al cada vez mayor incremento en el seguimiento de videojuegos, se empiezan a crear nuevas plataformas de streaming de estos. Dos de las principales plataformas actuales de transmisión son Youtube, y Twitch, centrada, sobre todo, en la transmisión en directo.
Nuestro proyecto ofrece la integración de estas dos plataformas en una sola web, para facilitar a los usuarios de ambas el que sea más fácil encontrar contenido de cualquier videojuego. Para ello usaremos las APIs de Twitch y Youtube, IGDB y Evernote, para guardar el enlace a un video para verlo más tarde.
